## Terminal

| Shortcut | Description                  |
| -------- | ---------------------------- |
| Ctrl + U | Delete typed row in terminal |
| Cmd + T  | New terminal tab             |

&nbsp;

| Command  | Description                |
| -------- | -------------------------- |
| Man      | Help pages (q to exit)     |
| mkdir -p | Create dir and parent dirs |

---

&nbsp;
